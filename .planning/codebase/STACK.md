# Technology Stack

**Analysis Date:** 2026-04-21

## Languages

**Primary:**
- TypeScript 5.7.2 - All source code, including main process, preload, renderer, and shared types
- JSX/TSX - React component definitions in renderer process

**Secondary:**
- JavaScript - Generated by TypeScript compilation

## Runtime

**Environment:**
- Node.js (version defined by host system, required for build and Electron processes)
- Electron 33.2.1 - Desktop application framework (main and renderer processes)

**Package Manager:**
- npm 10.x (implied by package-lock.json v3)
- Lockfile: `package-lock.json` (present)

## Frameworks

**Core:**
- Electron 33.2.1 - Desktop application runtime for multi-process architecture
  - Main process: Manages window lifecycle, IPC handlers, git operations, pty sessions
  - Renderer process: React-based UI
  - Preload: Context bridge for secure IPC
- React 19.0.0 - UI framework for renderer process
- Vite 6.0.3 - Build tool and dev server

**Terminal & Editor:**
- xterm.js 5.5.0 - Terminal emulation library
  - @xterm/addon-fit 0.10.0 - Fit addon for responsive terminal sizing
- @monaco-editor/react 4.6.0 - Code editor and diff viewer

**State Management:**
- zustand 5.0.2 - Lightweight state management (renderer process store at `src/renderer/store.ts`)

**Build Tools:**
- vite-plugin-electron 0.29.0 - Vite plugin for Electron main/preload bundling
- vite-plugin-electron-renderer 0.14.6 - Vite plugin for Electron renderer process
- @vitejs/plugin-react 4.3.4 - React support for Vite
- electron-builder 25.1.8 - Application packaging and installer creation

## Key Dependencies

**Critical:**
- simple-git 3.27.0 - Git operations (branch creation, worktree management, diff calculation, commit, push, PR creation)
- node-pty 1.0.0 - Pseudo-terminal management for spawning child processes (claude/codex CLI agents)

**Infrastructure:**
- @types/node 22.10.0 - TypeScript definitions for Node.js APIs
- @types/react 19.0.0 - TypeScript definitions for React
- @types/react-dom 19.0.0 - TypeScript definitions for React DOM

## Configuration

**Environment:**
- Development: Vite dev server serves renderer process; Electron loads from VITE_DEV_SERVER_URL
- Production: Built HTML served from `dist/index.html`; main process bundled to `dist-electron/main.js`
- Preload: Bundled to `dist-electron/preload.js`

**Build:**
- `vite.config.ts` - Vite configuration with:
  - React plugin for JSX/TSX
  - Electron plugin for main/preload bundling
  - Path alias: `@shared` → `src/shared/`
  - External dependencies: electron, node-pty, simple-git (not bundled in main process)
- `tsconfig.json` - TypeScript configuration:
  - Target: ES2022
  - Module: ESNext
  - JSX: react-jsx
  - Strict mode enabled
  - Path alias: `@shared/*` → `src/shared/*`

**Packaging:**
- electron-builder configured in `package.json` build section:
  - App ID: `dev.orchestra.app`
  - Output: `release/` directory
  - Targets: AppImage (Linux), DMG (macOS), NSIS (Windows)
  - Product name: Orchestra

## Platform Requirements

**Development:**
- Node.js runtime (build and execution)
- npm for dependency management
- Available git executable (for git operations and default branch detection)
- `claude` or `codex` CLI available in PATH (agent commands executed via node-pty in worktree)
- `gh` CLI available in PATH (GitHub PR creation)
- Supported terminals: xterm-256color environment
- Native C++ compilation tools (node-pty requires compilation via node-gyp)

**Production:**
- Desktop OS: Linux (AppImage), macOS (DMG), Windows (NSIS installer)
- Filesystem: Access to `~/.orchestra/worktrees/` for worktree storage
- Git: System git installation
- Agents: `claude` and `codex` CLI tools in PATH
- GitHub CLI: `gh` command for PR creation (optional, but required for PR feature)

## Build Output Structure

**Development:**
- Renderer: Served by Vite dev server on VITE_DEV_SERVER_URL
- Main/Preload: Watched by vite-plugin-electron, rebuild on changes

**Production:**
- `dist/` - Compiled renderer assets (HTML, CSS, JS)
- `dist-electron/main.js` - Bundled main process (CommonJS format)
- `dist-electron/preload.js` - Bundled preload script (CommonJS format)
- `release/` - Packaged installers (created by electron-builder)
